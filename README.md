# 🐍 Jornada Python: Lógica e Algoritmos

Este repositório é dedicado ao armazenamento de todos os meus estudos, exercícios e desafios práticos de Python. Ele reflete minha evolução tanto no **Curso Técnico** quanto nos meus **estudos autônomos online**.

O foco principal aqui é o domínio da lógica de programação e a resolução de problemas utilizando Python como ferramenta principal.

## 📂 Estrutura do Repositório

O projeto está organizado da seguinte forma:

*   **`curso/`**: Contém os desafios e exercícios práticos realizados no curso de Python online (incluindo scripts de conversão, cálculos e lógica).
*   **`ex-ete/`**: Exercícios e atividades desenvolvidas durante as aulas do Curso Técnico, focados em lógica de programação e fundamentos de algoritmos.

## 🚀 Tecnologias Utilizadas

*   **Python 3**: Linguagem principal para o desenvolvimento da lógica.
*   **Git & GitHub**: Versionamento de código e organização dos estudos.
*   **VS Code**: Editor de código utilizado para o desenvolvimento.
